# Mathemagicia

Мистическая математическая игра-головоломка для детей 6–10 лет в алхимическом стиле.

## О проекте

Mathemagicia — веб-приложение (PWA), где каждый уровень — это магическая задача. Четыре главы, каждая со своим типом головоломки:

- **I — Граф-цепочка** — проведи путь через узлы так, чтобы сумма чисел равнялась цели
- **II — Магический квадрат** — расставь числа так, чтобы суммы по строкам, столбцам и диагоналям совпадали
- **III — Алхимические фигуры** — перенеси каждую фигуру в её паз
- **IV — Созвездия** — соедини звёзды по порядку и проявь созвездие

Прогресс бесконечный — каждые 6 уровней открывают новое реальное созвездие из каталога IAU (88 созвездий). Все открытые созвездия собираются в коллекцию «Небо открытий».

## Стек

- React 18 (UMD, без бандлера)
- Babel Standalone (JSX прямо в браузере)
- SVG-анимации, CSS-переходы
- PWA (манифест + service worker)

## Запуск

Достаточно открыть `MathGame/index.html` в браузере — никакой сборки не требуется.

Для PWA-версии (с установкой на устройство) используй `WebGame/index.html` через локальный сервер:

```bash
cd WebGame
node server.js
```

Затем открой `http://localhost:3000` в браузере.

## Структура

```
MathGame/       — исходники игры
  prototype.jsx     оркестратор экранов
  screens.jsx       все экраны (меню, главы, победа, настройки)
  atoms.jsx         переиспользуемые компоненты и переводы
  puzzles.jsx       головоломки I и II глав
  puzzles-2.jsx     головоломки III и IV глав
  graph-gen.js      генератор уровней (xorshift PRNG)
  constellations-data.js  база 88 реальных IAU созвездий

WebGame/        — PWA-обёртка
GodotGame/      — прототип на Godot (архив)
```
